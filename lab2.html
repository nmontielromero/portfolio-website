Describe when to use the following HTML elements:

  • <br>: Used to define a table row.
  • <>: Used to define a table header cell(often used in the heading row ).
  • <>: Used to define a regular table data cell.
  • <thead>: Used to group the table header content.
  • <tfoot >: Used to group the table footer content.
  • <caption>: Used to provide a title or caption for the table.

What is the difference between colspan and rowspan?

colspan is an attribute used to specify the number of columns a cell should span. rowspan is an attribute that specifies the number of rows a cell should span.

What is the difference between cell spacing and padding?

Cell spacing refers to the space between cells in a table. Padding refers to the space between the content of a cell

How do you define table borders using CSS?

To specify table borders in CSS, use the border property.

How do you control border widths using CSS?

Using border-width: top_bottom left_right; values the widths are added, the first value will apply to the top and bottom of the box. The second value will apply to the left and right sides of the box.
